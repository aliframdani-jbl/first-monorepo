Usage: mager-commit [options]

An AI-powered commit generator tools. Provided for manusia-manusia mager.

Options:
  -V, --version  output the version number
  -d, --default  Run a commit generator with every params decided by the Generative AI.
  -c, --custom   Customize the commit message generator that will be generated by Generative AI.
  -h, --help     Show help.
```

## Setup

```
